Readme: Demo mode

To enable the demo mode, assign "true" for the "AppManager/is_demo_only" parameter in the config.ini file. Otherwise, it will run in the test mode automatically.

The demo/tests will run in a loop in the demo mode. It won't stop unless the "P" key on the keyboard is pressed.

Press "S" on the keyboard to start a demo.

Press "P" on the keyboard to force the demo to stop.

Press "M" on the keyboard / Hand Trigger on the controller to change to the next SoundVizVR mode.
